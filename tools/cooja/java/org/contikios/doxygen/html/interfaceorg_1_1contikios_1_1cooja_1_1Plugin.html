Collection<Element> org.contikios.cooja.Plugin.getConfigXML ( )

Returns XML elements representing the current config of this plugin. This is fetched by the simulator for example when saving a simulation configuration file. For example a plugin may return the current size and position. This method should however not return state specific information such as the value of a mote LED, or total number of motes. (All nodes are restarted when loading a simulation.)
