A simulated mote.

All motes have an interface handler, a mote type and a mote memory.

abstract Collection<Element> org.contikios.cooja.Mote.getConfigXML ( )
abstract

Returns XML elements representing the current config of this mote. This is fetched by the simulator for example when saving a simulation configuration file. For example a mote may return the configs of all its interfaces. This method should however not return state specific information. (All nodes are restarted when loading a simulation.)
